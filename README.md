## Description
This project is a single-page CV built using semantic HTML and styled with basic CSS. It presents education, skills, projects, and experience in a clean and structured layout.

## Features
- Semantic HTML structure
- SEO meta tags
- Open Graph (OG) tags
- Favicon integration
- Clean and responsive layout
- Deployed using GitHub Pages

## Technologies Used
- HTML5
- CSS3

## Sections Included
- About Me
- Education
- Skills
- Projects
- Experience
- Contact
